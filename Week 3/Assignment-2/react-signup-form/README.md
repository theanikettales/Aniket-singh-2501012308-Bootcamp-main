# React Signup Form

A simple, validated signup form built with React and Vite.

## Features
- Controlled inputs for name, email, password, and confirm password
- Client-side validation (required fields, email format, password length, matching passwords)
- Success screen after a valid submission

## Getting started

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal (usually http://localhost:5173).

## Build for production

```bash
npm run build
```
