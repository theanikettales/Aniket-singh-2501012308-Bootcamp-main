# Cricket Score Board (React)

A React + Vite version of the cricket scoreboard app. Simulates one over
(6 balls) — each click on "Click to hit the ball" adds a random run value
(0, 1, 2, 3, 4, or 6) to the score. The button disables once 6 balls have
been bowled, and "Reset Game" starts a new over.

## Getting started

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal (usually http://localhost:5173).

## Build for production

```bash
npm run build
```
